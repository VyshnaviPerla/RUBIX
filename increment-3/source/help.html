HOW TO COMPOSE MAILS 



 click on compose mail option write
 sender's id in 'TO'box. write subject 
 in 'SUBJECT' box.write message in 'MESSAGE'
 box.you can send mail on specified date by
 mentioning date or you can send now itself.
 back to top
HOW TO READ RECEIVED MAILS 



 click on inbox option there you have read mail 
 as well as unread mails here you can read mails
 or delete mails .
 back to top
HOW TO DELETE MAILS 



 click on inbox option from there you can delete 
 mails and that mails goes to trashcan.
 back to top
HOW TO DELETE MAILS PERMANENTELY 



 click on trash option select mails which
 you want to delete  by clicking checkbox 
 now click delete button.that mail is 
 deleted permanently.
 back to top
HOW TO RESTORE MAIL AGAIN TO INBOX 



 click on trash option select mails which
 you want to restore back to inbox  by 
 clicking checkbox now click restore button.
 back to top
HOW TO SEARCH MAIL 



click on search mail option to search mail 
write either sender's id or any specific 
pattern in the subject.click search button.
back to top
HOW TO RECOVER PASSWORD 



 click on forgot password option in the home 
 page.write your login id in 'username' box
 then answer the quesion.now you are allowed
 to give new password.
back to top
